== Description ==

InterSlider is a responsive and customizable slider made in jQuery.

See <a href="http://interkod.pl/interslider/">working demo</a>.

== Installation ==

1. Download plugin and unpack it.
2. Upload `inter-slider` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. There are two ways of puting slider on your site 
	* Insert `<?php do_action('inter_slider'); ?>` in your template
	* Go to Appearance -> Widgets and drag `InterSlider` widget to the sidebar of your chose
